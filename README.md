A simple Arduino IDE sketch to use a Pi Pico 2 W as an ammeter with a local display
and a web page hosted via wifi for remote checking.

This is designed to be easily expanded to include features like:
*  Sending data via syslog to a remote server or save locally on an SD card
*  Integration with HomeKit or other automation systems
*  Addition of external alarms or indicators for various error conditions
*  Calculation of coulombs to track Wh or Ah usage

If you add functionality to the sketch and want to merge it back into this project,
feel free to submit a PR or send me your code for review.  You're also welcome to fork
this repo to use as you see fit.
